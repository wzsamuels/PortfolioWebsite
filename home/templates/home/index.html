Ticket Manager ×

The Ticket Manager program handles a list of support technical support tickets.

Due to NCSU's academic integrity policy, the project's github repository is private, but it can be made available to potential employers upon request.

Technical and Design Concepts:

  • Written in Java.
  • The State Design Pattern: each ticket object changes state as it's worked on.
  • MVC Design Pattern for the interaction of the backend and GUI.
  • Polymorphism: a ticket's state change is reflect by a change in its object type.

DataStructures ×

DataStructures is a collection of data structures and sorting algorithms that can be imported into other projects as a dynamic-link library. Adapted from a java project started at NCSU, the code has been completely ported to C# to comply with NCSU's academic integrity policies.

Github repository available here.

Technical Concepts:

  • Written in C#
  • Unit tests for all data structures and algorithms
  • Custom implementations of many data structures, including:
    • Hash Tables:
      • Linear Probing Hash Map
      • Separate Chaining Hash Map
    • Lists:
      • Array-based List
      • Positional Linked List
      • Singly Linked List
    • Maps:
      • Search Table
      • Skip List
      • Unordered Array
      • Unordered Linked List
    • Search Trees:
      • AVL Tree
      • Binary Search Tree
      • Red Black Tree
      • Splay Tree
  • Custom implementations of the following sort algorithms:
    • Bubble Sort
    • Counting Sort
    • Insertion Sort
    • Merge Sort
    • Quick Sort
    • Radix Sort
    • Selection Sort
